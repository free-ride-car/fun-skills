# 使用示例

本文档提供 Chrome Browser Skill 的完整使用示例。

## 基础示例

### 示例 1: 打开网站

**用户**: 打开 GitHub 首页

**执行流程**:
```
1. 检查浏览器连接 → 已连接
2. navigate → https://github.com
3. 等待页面加载
4. 返回页面信息
```

**助手响应**:
> 已打开 GitHub 首页
> - 标题: GitHub: Let's build from here
> - URL: https://github.com

---

### 示例 2: 搜索

**用户**: 在 Google 搜索 "Chrome DevTools Protocol"

**执行流程**:
```
1. navigate → https://www.google.com
2. wait → 搜索框出现
3. type → input[name="q"] → "Chrome DevTools Protocol"
4. press → Enter
5. wait → 搜索结果加载
6. screenshot → 返回截图
```

**助手响应**:
> 已完成搜索，找到约 2,340,000 条结果
> [搜索结果截图]

---

### 示例 3: 获取页面数据

**用户**: 获取当前页面的所有标题

**执行流程**:
```
evaluate →
Array.from(document.querySelectorAll('h1, h2, h3'))
  .map(h => ({ level: h.tagName, text: h.textContent.trim() }))
```

**助手响应**:
> 页面标题列表：
> - H1: Welcome to Our Site
> - H2: Features
> - H2: Pricing
> - H3: Basic Plan
> - H3: Pro Plan

## 进阶示例

### 示例 4: 表单填写

**用户**: 帮我在登录页面填写账号密码，账号是 test@example.com

**执行流程**:
```
1. 确认当前页面是否有登录表单
2. evaluate → 查找表单元素
3. type → input[type="email"] → "test@example.com"
4. type → input[type="password"] → "******" (询问密码)
5. 等待用户确认后点击登录
```

**助手响应**:
> 找到登录表单，已填写邮箱。
> 请问密码是什么？填写后我可以帮你提交登录。

### 示例 11: 浏览器未连接

**用户**: 打开百度

**执行流程**:
```
1. 尝试连接浏览器
2. 连接失败
3. 提供启动指南
```

**助手响应**:
> 无法连接到 Chrome 浏览器。
>
> 请先启动 Chrome 并开启远程调试：
>
> **Windows**:
> ```
> "C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222
> ```
>
> **macOS**:
> ```
> /Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222
> ```
>
> 启动后请告诉我，我会继续操作。
